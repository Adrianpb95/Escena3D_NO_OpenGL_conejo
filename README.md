# Escena3D_NO_OpenGL_conejo
Creación de escena 3D, sin usar librería gráfica. (C++ _ TinyObj)

El objetivo de este proyecto es la creación de un software capaz de cargar escenas 3D y realizar diversas transformaciones en ellas, sin la ayuda de librerias gráficas externas, pero basándose en sus mismos principios: vertex buffer, color buffer, z buffer, iluminación y renderizado de escenas.

Para abrir el ejecutable dirigirse a la carpeta: 4b - z-buffer\binaries                                                                   
Y abrir el archivo "Z-Buffer"

Para abrir el proyecto dirigirse a la carpeta: 4b - z-buffer\projects\vs-2015                                                             
Y abrir la solución llamada: "Z-Buffer.sln"                                                                                               
Asegurarse de usar un SDK compatible (click derecho sobre la solución "Z-Buffer" > propiedades > versión del SDK de windows) y darle a compilar.
